# Конфигурации Nix

Этот репозиторий содержит конфигурации Nix для различных хостов и пользователей.

## Структура

```
.
├── flake.nix                 # Основной файл конфигурации флейка
├── dotfiles/                 # Конфигурации для различных программ
├── hosts/                    # Конфигурации для разных хостов
│   ├── l2/                   # Хост l2 (не настроен)
│   ├── s1/                   # Хост s1 (не настроен)
│   └── zerg/                 # Хост zerg (основной хост)
│       ├── configuration.nix # Основная конфигурация системы
│       ├── hardware-configuration.nix # Конфигурация оборудования
│       ├── default.nix       # Дефолтная конфигурация
│       ├── amd-cpu.nix       # Настройки AMD CPU
│       ├── nvidia.nix        # Настройки NVIDIA
│       └── dev-tools.nix     # Инструменты разработчика
├── modules/                  # Модули для различных целей
│   ├── nixos/                # Модули NixOS
│   │   └── nix.nix           # Настройки Nix
│   ├── profiles/             # Профили (laptop, server, desktop)
│   │   ├── laptop.nix
│   │   ├── server.nix
│   │   └── desktop/
│   │       ├── desktop.nix
│   │       ├── general.nix
│   │       ├── greetd.nix
│   │       ├── stylix.nix
│   │       └── wm/
│   │           ├── niri.nix
│   │           └── wms.nix
│   └── wrappedPrograms/      # Обернутые программы
│       └── env.nix
└── users/                    # Пользовательские конфигурации
    ├── alex/
    │   └── home.nix
    ├── alisa/
    │   └── home.nix
    └── zerg/
        └── home.nix
```

## Хосты

- `zerg`: Основной хост с полной конфигурацией
- `l2`: Пустой хост (требует настройки)
- `s1`: Пустой хост (требует настройки)

## Пользователи

- `zerg`: Основной пользователь
- `alex`: Второй пользователь
- `alisa`: Третий пользователь

## Установка

Для установки конфигурации:

```bash
nixos-rebuild switch --flake .#zerg
```

Для обновления конфигурации:

```bash
nix flake update
```

## Дополнительно

- `dotfiles/` содержит конфигурации для различных программ
- `modules/profiles/` содержит общие профили для разных типов систем
- `modules/wrappedPrograms/` содержит обернутые версии программ
