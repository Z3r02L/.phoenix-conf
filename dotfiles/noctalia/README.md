# Noctalia Shell Configuration
# This folder is symlinked to ~/.config/noctalia

Noctalia is a modern, high-performance Wayland shell built specifically for the Mango WM ecosystem. It is based on **Quickshell** and provides a sleek environment including a top bar, system notifications, and desktop widgets.

## Components
- **noctalia-shell**: The main panel and widget engine.
- **noctalia-wallpaper**: Integrated wallpaper management.

## Integration with Mango WM
The Mango configuration in `dotfiles/mango` is set up to automatically launch Noctalia component on startup.
